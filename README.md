# 工数計算アプリ
Pythonを勉強してはじめて2週間の成果物です。
Streamlitで作成したシンプルな計算ツールです。
作業時間から生産台数と数をかけたものを割って、秒数を計算してくれます。
仕事で配線作業をしていて、計算するのが面倒だったことから生まれたアイデアです。

## 機能
-機種選択　(A,B,C)
-生産台数・数の入力
-作業時間(分)の入力
-1本あたりにかかる秒数を計算
-計算結果の保存

'''bash
streamlit run work_app.py
